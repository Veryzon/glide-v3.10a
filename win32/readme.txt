+--------------------------------------------------------------+

 The Glide Open Source Project - 3dfx Glide3x driver for win32

+--------------------------------------------------------------+

Version      : 3.10.00.30303

Requirements : Voodoo Banshee,Velocity 100/200,Voodoo3/4/5
               Microsoft Windows 95/98/Me/NT4/2000/XP
               Windows driver from 3dfx Interactive Inc.
               DirectX 8.1a or newer runtime

Installation : Backup glide3x.dll and copy the new file to the following directory;
               Windows 95/98/Me : \Windows\System
               Windows NT4/2000 : \Winnt\System32
               Windows XP       : \Windows\System32

Note         : glide3x.lib  : MSC import library.
               libglide3x.a : MinGW import library.

Web Site     : http://glide.sourceforge.net/
